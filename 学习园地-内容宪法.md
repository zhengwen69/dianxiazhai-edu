# 学习园地 · 内容宪法

> 版本：v1.0 | 生效日：2026-05-25
> 适用范围：`学习-第1周.html` 至 `学习-第8周.html` 全部学习页面
> 宪法属性：后续任何内容修订、新增、替换，必须逐条对照本标准执行，不得偏离。

---

## 第一条：身份标识原则

凡引用教学材料出处，一律使用规范称谓：

| 场景 | 规范表述 | 禁用表述 |
|------|----------|----------|
| 讲义引用 | **黄正文教授自编讲义 §X** | "教材""课本""讲义"（无归属） |
| 小说引用 | **纪实小说《XXX》教学选段（S0XX）** | "案例材料""补充阅读" |
| 方法论 | **黄正文教授 · 错配-修复框架** | "分析方法""理论框架"（无归属） |
| 教学法 | **叙·框·境·创 四维教学法** | （全程必用，不得省略） |
| 作者署名 | **点暇叙事 匠心教学 · 黄正文（点暇斋）** | 仅出现在页面尾部 |

## 第二条：本科前置不占用原则

每周"课前预习"第一张卡片必须为**前置自检**，以红色 `.precheck` 样式呈现。内容为本科已修但研究生不再重复讲授的知识点，`.precheck` 正文必须包含短语"本科已修，此处不重复讲授""如有模糊请自行复习"。

**第8周豁免**：第8周为结课整合周，前置自检改为"全部学习完成确认"，内容为第1-7周学习任务完成确认及结课产出方向初步确定。

**禁止**将以下本科层次内容作为研究生教学内容正文展开：
- 固废定义/分类/来源/特性
- 焚烧/热解/气化基本原理
- 好氧堆肥/厌氧消化基本原理
- 垃圾分类四分法
- EPR基本概念
- 环评基本制度/NIMBY基本概念

## 第三条：六大标识元素强制穿透

每周页面四个区域（预习/学习/作业/AI赋能）必须全覆盖以下六大元素：

| 编号 | 元素 | 要求 | 出现位置 |
|------|------|------|----------|
| ① | 黄正文教授自编讲义章节号 | 必须出现 §X.X | 预习区 |
| ② | 纪实小说选段编号 + 原文引用 | 必须出现 S0XX + `.excerpt` 引用块 | 预习区 + 学习区 |
| ③ | 错配-修复框架子维度 | 必须出现带标签的维度列表 | 学习区 |
| ④ | 叙·框·境·创 主导维度标签 | 必须出现黄色 `tag` + 文字说明 | 学习区第一卡片 |
| ⑤ | 方法论语块 + 研究空白 | 必须出现紫色 `.method` 卡片 | 学习区末卡片 |
| ⑥ | 专属 Prompt（含黄正文专有术语） | 每 Prompt 必含框架/术语 + `.prompt` 样式 | AI赋能区 |

## 第四条：叙事结构规范

### 4.1 案例深潜区

每周学习区必须包含"案例深潜"卡片，对应当周纪实小说选段进行**五段式叙事拆解**：

```
● 背景：XXX（一句话背景设定）
● 冲突：XXX（核心矛盾是什么）
● 决策：XXX（面临的选择/博弈）
● 结局：XXX（发生了什么）
● 启示：XXX（从叙事到分析的升华——必含黄正文教授核心判断）
```

**第8周豁免**：第8周为综合整合周，案例深潜以"全回顾视角"替代单周五段叙事。内容为S001-S021七部小说全景回顾及八周叙事进化总结。

### 4.2 引用块格式

小说原文引用使用 `.excerpt` 样式，包含作者、作品、选段编号、原文，以斜体浅色呈现，左侧彩色边框色与当周主题色一致。

## 第六条：叙·框·境·创 八周分布

| 周次 | 主导维度 | 教学重心 |
|------|----------|----------|
| 第1周 | **叙** | 以叙事触发感性认知，从故事引出学术概念 |
| 第2周 | **框** | 将感性认知升华为可操作的分析工具 |
| 第3周 | **境** | 以场景模拟（环评答辩）为课堂情境 |
| 第4周 | **境→创** | 从场景分析转向创造性方案设计 |
| 第5周 | **境** | 以乡村分类失败场景为课堂情境 |
| 第6周 | **框→境** | 框架分析 + 制度设计场景模拟 |
| 第7周 | **境** | 模拟法庭/听证会为课堂场景 |
| 第8周 | **创** | 综合研讨 + 模拟决策 + 三轨产出 |

## 第七条：AI赋能标准

### 7.1 Prompt 铁律

每条 AI Prompt 必须包含**至少一个**黄正文教授专有术语，确保不可替代性：

| 维度 | 每周 Prompt 核心术语 |
|------|---------------------|
| 空间 | "空间错配五维框架""选址正义性四维""空间正义" |
| 技术 | "技术错配三型""垃圾时间三维""技术适用性矩阵" |
| 行为 | "行为错配四因""最后一米困境""EPR错配三源""激励相容" |
| 人文 | "人文错配三维""社会许可证""承认正义" |
| 综合 | "五维统一诊断矩阵""叙·框·境·创""四重错配" |

### 7.2 Prompt 设计标准

- 引导 AI 产出"分析方法"而非"知识总结"
- 每个 Prompt 至少可对接一个 Agent（Agent-S/I/4/B）
- 使用 `.prompt` 样式（黑色背景 + 等宽字体）

## 第八条：第8周 · 结课产出三轨制

### 轨道A：课程论文
- 运用"五维统一诊断矩阵"（至少三维）
- 8000字以内
- 必须标注框架 + 方法论语块 + 讨论"第六维"

### 轨道B：课程设计
- 填埋场/焚烧厂/堆肥厂 三选一
- 空间+技术必选 + 行为/人文/时间任选一
- 产出：说明书 + 流程图 + 参数计算 + 三维论证

### 轨道C：教改论文
- 叙框境创效果分析 / 小说云驱动接受度 / 学习子门户可用性
- **必须**引用、体现、联系黄正文教授有关《固体废弃物污染防治技术》课程有且不仅限于叙框境创教育教学教研教改的理念文本
- 5000-8000字

## 第九条：CSS样式规范

| 元素 | 类名 | 作用 |
|------|------|------|
| 前置自检 | `.precheck` | 红色背景提示 |
| 小说引用 | `.excerpt` | 斜体 + 左侧边框（色随周变） |
| AI Prompt | `.prompt` | 黑底 + 等宽字体 |
| 方法论语块 | `.method` | 紫色背景 |

| 框架维度 | `.tag-{color}` | 彩色小标签 |
| 结课轨道 | `.track-label .track-{a/b/c}` | 彩色轨道标识 |
