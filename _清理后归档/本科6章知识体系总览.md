## 一、课程总体框架

本科《固体废物处理与处置》课程共设**6大知识模块**，覆盖固废从"源头"到"末端"的全生命周期管理：

| 序号 | 章节名 | 教材对应章 | 教学周 | 周课时 | 核心主题 |
|:---:|--------|:---------:|:-----:|:-----:|---------|
| 1 | 绪论 | 第1章 | 第1周 | 2 | 定义、分类、特性、法规、管理体系 |
| 2 | 收集运输与预处理 | 第2~4章 | 第2~4周 | 6 | 采样分析、收运系统、转运站、破碎分选 |
| 3 | 生物处理 | 第5~6章 | 第3~4周 | 4 | 好氧堆肥、厌氧消化（沼气） |
| 4 | 热处理 | 第7~9章 | 第4~5周 | 4 | 焚烧、热解、气化 |
| 5 | 填埋处置 | 第11章 | 第7~8周 | 4 | 卫生填埋、渗滤液、填埋气 |
| 6 | 资源化 | 第10章 | 第6周 | 4 | 废纸/塑料/橡胶/冶金渣/秸秆回收利用 |

## 三、章节依赖关系

```
第1章 绪论（基础概念、分类、法规）
  │
  ├──→ 第2章 收集运输预处理（采样、收运、破碎分选）
  │      │
  ├──────┼──→ 第3章 生物处理（堆肥、沼气）
  │      │      └── 有机废物 + 预处理
  │      │
  ├──────┼──→ 第4章 热处理（焚烧、热解、气化）
  │      │      └── 可燃废物 + 热值 + 预处理
  │      │
  ├──────┼──→ 第6章 资源化（材料回收、建材化、能源化）
  │      │      └── 可回收废物 + 分选
  │      │
  └──────┴──→ 第5章 填埋处置（最终安全处置）
               └── 所有工艺的末端残渣去向
```

## 五、核心公式与计算要点汇总

| 章节 | 核心公式/方法 | 应用场景 |
|------|-------------|---------|
| 第2章 | 采样间隔 = 批量/份样数 | 固废采样布点 |
| 第2章 | 每车应采份样数 = 规定份样数/车数 | 运输车采样 |
| 第2章 | C = (运输成本)/(容积×60)×t + 管理费 | 转运站经济性判断 |
| 第2章 | Nw = 垃圾量/(车容积×利用系数×压缩比) | 清运次数计算 |
| 第3章 | 堆肥物料含水率调节：G₁×W₁ + G₂×W₂ = (G₁+G₂)×W_target | 堆肥含水率调配 |
| 第3章 | 每立方米堆层通风量 0.1~0.2 m³/min | 堆肥曝气设计 |
| 第3章 | 沼气理论产气量 0.63~1.0 m³/kg VS | 厌氧消化产气估算 |
| 第4章 | 理论空气量 V₀ = 按元素组成化学计量计算 | 焚烧空气供给设计 |
| 第4章 | 实际空气量 V = α × V₀ (α为过量空气系数) | 焚烧炉运行 |
| 第4章 | 烟气停留时间 t = V/Q ≥ 2s (>850°C) | "3T+E"设计 |
| 第5章 | 单位面积渗透量 q = K×Δh/L | 填埋场渗滤估算 |
| 第5章 | 填埋气产气量 Q = 填埋量 × 单位产气率 × 产气年限 | 气体收集系统设计 |
| 第5章 | 发电容量 P = Q × CH₄含量 × LHV × 发电效率 | 填埋气发电设计 |
