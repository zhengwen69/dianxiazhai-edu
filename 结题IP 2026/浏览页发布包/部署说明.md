## 操作步骤

### 1. 复制文件到仓库

将以下文件复制到 `cdu-gufei-web-demo` 仓库对应位置：

- `结题报告.html` → 仓库根目录
- `结题报告.pdf` → 仓库根目录
- `assets/img/report/` 整个文件夹 → 仓库 `assets/img/report/`

### 2. 修改 index.html

在 `index.html` 中找到 **研究进展** 区域（搜索 `教研科研进展`），在其后新增一张卡片：

```html
<a href="结题报告.html" class="card">
  <span class="icon">📋</span>
  <h3>IP孵化项目结题报告</h3>
  <p>从产品IP到方法论IP · 10章 · 28表 · 6图 · 全文在线 · PDF下载</p>
</a>
```

### 3. 推送

```bash
git add .
git commit -m "add: IP孵化项目结题报告 全文在线浏览+PDF下载"
git push origin main
```

### 4. 验证

访问 https://zhengwen69.github.io/cdu-gufei-web-demo/结题报告.html 确认页面正常。

## 功能

- 全文HTML在线浏览（侧栏章节导航）
- 28张三线表（CSS渲染）
- 6张图表（PNG嵌入）
- PDF一键下载按钮
- 暗色主题，适配移动端
