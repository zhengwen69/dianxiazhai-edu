## 三、研究/设计目标与内容

### 3.1 核心问题

本设计聚焦八周教学交互式时间线，拟回答/完成以下问题/任务：

1. 完成"八周教学交互式时间线"的完整交互式可视化设计
2. 实现核心交互功能（展开/折叠）
3. 确保页面与现有固废课程门户的视觉风格一致

### 3.2 研究方法/技术路线

- **方法/技术**: HTML + CSS + JavaScript（时间线展开交互）
- **数据/素材**: 专著章节 Ch8 全程 中的内容数据
- **分析工具/开发环境**: VS Code / Chrome DevTools
- **核心工作**: 提取八周教学节点数据 → 设计纵轴时间线 → 实现展开/折叠交互 → 适配多语种 → 测试优化
- **独立工作内容**: 独立提取教学时间节点数据、独立设计时间线交互逻辑、独立完成前端编码
