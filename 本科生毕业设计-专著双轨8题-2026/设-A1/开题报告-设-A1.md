## 三、研究/设计目标与内容

### 3.1 核心问题

本设计聚焦五垃圾标签交互式可视化，拟回答/完成以下问题/任务：

1. 完成"五垃圾标签交互式可视化"的完整交互式可视化设计
2. 实现核心交互功能（树形展开/搜索筛选）
3. 确保页面与现有固废课程门户的视觉风格一致

### 3.2 研究方法/技术路线

- **方法/技术**: HTML + CSS + JavaScript（树形展开交互）
- **数据/素材**: 专著章节 Ch3 §3.4 中的内容数据
- **分析工具/开发环境**: VS Code / Chrome DevTools
- **核心工作**: 解析五垃圾标签体系 → 设计树形展开交互 → HTML/CSS/JS编码 → 适配8语种 → 测试优化
- **独立工作内容**: 独立设计信息架构与交互逻辑、独立完成全部前端代码编写
