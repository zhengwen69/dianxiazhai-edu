## 三、研究/设计目标与内容

### 3.1 核心问题

本设计聚焦三轨制作品集展示页，拟回答/完成以下问题/任务：

1. 完成"三轨制作品集展示页"的完整交互式可视化设计
2. 实现核心交互功能（分类筛选）
3. 确保页面与现有固废课程门户的视觉风格一致

### 3.2 研究方法/技术路线

- **方法/技术**: HTML + CSS + JavaScript（分类筛选展示）
- **数据/素材**: 专著章节 Ch8 §8.8 中的内容数据
- **分析工具/开发环境**: VS Code / Chrome DevTools
- **核心工作**: 整理三轨制产出样例 → 设计分类展示 → 实现筛选/展开交互 → 适配多语种 → 测试优化
- **独立工作内容**: 独立整理三轨制产出样例数据、独立设计展示交互逻辑、独立完成前端编码
