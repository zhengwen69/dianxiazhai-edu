### 二、投稿操作流程 / Submission Workflow

#### Step 1: 注册账号 / Register

1. 访问 https://www.sajournalofeducation.co.za/index.php/saje/user/register
2. 填写注册信息（姓名、邮箱、所属机构）
3. 验证邮箱

#### Step 2: 登录投稿系统 / Login

1. 访问 https://www.sajournalofeducation.co.za/index.php/saje/login
2. 使用注册账号登录
3. 点击 "New Submission" 开始投稿

#### Step 3: 上传文件（按此顺序） / Upload Files (in this order)

| # | 文件 | 格式 | 说明 |
|:--:|------|:--:|------|
| **1** | `P4-EN-SAJE-正文-v1.3.docx` | Word | **匿名正文**（无作者信息）：含 Abstract / §1–§6 / References / Tables / Figures / Declaration |
| **2** | `P4-EN-SAJE-Title-Page-v1.3.docx` | Word | **标题页**（独立文件）：含作者、单位、基金、伦理声明、CRediT、通讯信息 |
| **3** | `Figure1-知识域映射轨迹.png` | PNG | 图 1：知识域映射轨迹（≥300 dpi） |
| **4** | `Figure2-迁移探索枢纽页.png` | PNG | 图 2：迁移探索枢纽页截图 |
| **5** | `Figure3-无人机应用技术页面.png` | PNG | 图 3：无人机应用技术子页截图 |
| **6** | `P4-EN-SAJE-Highlights-v1.3.docx` | Word | 5 条 Highlight（每条 ≤85 字符） |
| **7** | `Supplementary-Material.pdf` | PDF | 辅助案例摘要数据（需另行制作） |

#### Step 4: 粘贴 Cover Letter / Paste Cover Letter

在系统 "Cover Letter" 文本框中粘贴以下内容（`P4-EN-SAJE-Cover-Letter-v1.3.docx` 的内容）：

> Dear Editors,
> 
> We submit the above-titled manuscript for consideration...（完整内容见 Cover Letter 文件）

#### Step 5: 填写元数据 / Fill Metadata

| 字段 | 内容 |
|------|------|
| Title | Cross-Disciplinary Teaching in Vocational Colleges: A Self-Directed Model for Teacher Professional Growth |
| Abstract | （粘贴正文摘要，≤250 词） |
| Keywords | cross-disciplinary teaching; self-directed professional development; vocational education; technology-assisted learning; teacher professional growth; scarce-resource contexts; BRICS education cooperation |
| Article Type | Research Article |
| Language | English |

#### Step 6: 提交 / Submit

1. 确认所有文件已上传
2. 确认元数据已填写
3. 点击 "Submit"
4. 获取稿件编号（Manuscript ID）

#### Step 7: 等待审稿 / Await Review

- 初审（格式合规检查）：1–2 周
- 双盲送审：4–7 个月
- 审稿结果通知：通过系统邮件

### 四、投稿后跟进 / Post-Submission Follow-up

| 时间节点 | 操作 |
|------|------|
| 提交后 1 周 | 确认收到稿件编号 |
| 提交后 2 周 | 如未收到格式审查结果，联系编辑部（kjss@ku.th — 注意：SAJE 邮箱需确认） |
| 4–7 个月 | 等待双盲评审结果 |
| 收到审稿意见 | 逐条回复 → 提交修改稿 + Response to Reviewer |
