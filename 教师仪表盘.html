🔧 首次配置指引（只需做一次）

  1. 打开 formspree.io，点击右上角 Sign In，用 GitHub 账号登录。
  2. 点击 + New Form，创建 3 个表单：
     ① 「学生签到」— 命名随意
     ② 「周进度问卷」— 命名随意
     ③ 「同行评价」— 命名随意
  3. 每个表单创建后，复制其 Endpoint URL（格式为 https://formspree.io/f/xxxxx）。
  4. 打开 互动社区.html，找到 3 处 YOUR_FORMSPREE_XXX_URL，替换为实际的 Endpoint URL。
  5. 打开本文件 教师仪表盘.html，将 3 个卡片链接 YOUR_FORMSPREE_DASHBOARD_URL 替换为你的 Formspree 后台地址（https://formspree.io/forms）。
  6. 将修改后的 HTML 文件上传到 GitHub Pages 仓库，刷新网站即可使用。

💡 Formspree 免费版每月 50 条提交，对于小班教学足够。如需更多可升级付费版。
