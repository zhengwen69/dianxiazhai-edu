Agent 1 · 资料归档与标准化
Archivist — 资料管理 & 目录维护
统一文件命名规范，维护目录结构，管理版本历史。定期巡检生成「资料缺失清单」，驱动其他 Agent 按需产出。
文件重命名 + 版本归档
教学资料索引.md 维护
每月缺失检测 & 完整性报告
重复文件检测 & 合并建议
产出岗
Agent 2 · 教学文档生成
Document Builder — 大纲·日历·教案
基于 OBE 框架生成标准化教学文档：大纲（含课程目标-毕业要求矩阵）、16周教学日历、详细教案。自动校验目标-内容-考核闭环。
OBE 教学大纲（含矩阵）生成
16 周教学日历编排
教案/讲稿标准化撰写
课程思政要素自动嵌入
产出岗
Agent 3 · PPT 与讲稿设计
Lecture Designer — 课件·演示·HTML
按章节生成详细讲稿（导入→讲授→小结→思考题）和 PPT 内容框架，支持输出课堂演示 HTML 页面（带动效，16:9适配）。
讲稿撰写（含板书设计）
PPT 内容框架生成
HTML 演示页面（fade/slide 动效）
课程思政融入点设计
产出岗
Agent 4 · 作业与课程设计
Assignment Designer — 题库·试卷·课设
设计分层作业（基础/应用/拓展）、A/B 双卷（含参考答案+评分标准 Rubric）、工程课程设计任务书（填埋场/焚烧厂/堆肥厂）。
章节作业题库（分层难度）
A/B 卷 + 参考答案 + 评分标准
课程设计任务书（填埋场/焚烧/堆肥）
Rubric 评分量表自动生成
分析岗
Agent 5 · 教研教改与质量监控
QA & Reform — 达成度·反思·教改
基于成绩数据计算课程目标达成度，撰写教学反思与持续改进报告，辅助教改项目申报。形成「评价→反馈→改进」闭环。
课程目标达成度计算与分析
教学反思 + 持续改进建议
教改项目申报书撰写
工程教育认证对齐检查

Agent-S · 故事化案例拆解
Case Analyst — 工程→叙事
从云端已发布的纪实小说（《龙栖湾》QQ阅读、《青龙湖》《杨柳坝与刘家湾》《三多里巷》等17K'普惠教育咨询·读书改变命运秘笈'系列）中提取教学选段，转化为五段式教学叙事（背景→冲突→决策→结局→启示），每案附方法论语块与研究空白。
云端素材提取 (QQ阅读/17K→教学选段)
错配-修复框架匹配
分层讨论题配套(5题)
方法论语块+研究空白标注
