### "教学-学习"双重数字门户的协同设计与实践

| **摘要** | 针对高校课程网站普遍存在的"教师展示多、学生入口少"的单向设计缺陷，提出"教学-学习"双重数字门户协同架构。该架构由教师教学演示门户（课程概览、章节导航、知识导图、教学团队、互动社区、教师仪表盘）与学生学习子门户（学习园地）两部分组成，两者通过零配置纯静态技术栈（HTML+CSS+JS）实现GitHub Pages同步部署。文章以成都大学《固体废弃物污染防治技术》研究生精品课程为案例，阐述了双重门户的设计哲学（教管分离、以学为中心、零配置部署）、信息架构（三折叠导览、八周学习计划网格、四段式学习页）与协同机制（教师端产出→学生端消费的双向信息映射）。研究提出，从"单一教师门户"到"教-学双门户"的架构跃迁，是课程网站从"展示教学材料"到"支撑学习过程"的范式转换的关键一步。 |

### AI Agent协同体系在研究生教学中的角色重塑——从"教"到"伴学"的双向映射
#### 以《固体废弃物污染防治技术》课程为例

| | |
|---|---|
| **作者** | 王鸿斌¹，谢泽宇²，黄正文¹*（\*通讯作者） |
| **单位** | （1. 成都大学 建筑与土木工程学院 环境工程系，成都 610106；2. 西那瓦国际大学 Shinawatra University，泰国） |
| **摘要** | 人工智能赋能教育正从单点工具（查资料、写摘要、做PPT）向协同化的Agent体系演进。然而，当前的AI教育工具多服务于"教"——辅助教师备课、出题、批改——而面向"学"的Agent设计尚未得到系统性的关注。本文以成都大学研究生课程《固体废弃物污染防治技术》的9-Agent协同教学体系为案例，系统阐述Agent从"教"到"伴学"的角色重塑方法论。该体系在教师端定义了9个Agent的协同工作流——从素材归档（Agent-1）到教学文档生成（Agent-2）、PPT与讲稿设计（Agent-3）、作业与课设（Agent-4）、教研教改（Agent-5），再到领域特色Agent——故事化案例拆解（Agent-S）、场景沉浸设计（Agent-I）与学术品牌建设（Agent-B），全部产出经Agent-0终审。本文提出"教师→学生"双向角色映射方法论：教师端的9个Agent通过"镜像而非简化""服务对象翻转""角色自主重定义"三条原则，在学生端被重塑为9个伴学Agent，形成从"教"到"伴学"的完整映射链。 |

### 纪实小说云驱动——工程案例教学的故事化重构路径
#### 以《固体废弃物污染防治技术》研究生课程为例

| | |
|---|---|
| **作者** | 谢泽宇¹，王鸿斌²，黄正文²*（\*通讯作者） |
| **单位** | （1. 西那瓦国际大学 Shinawatra University，泰国；2. 成都大学 建筑与土木工程学院 环境工程系，成都 610106） |
| **摘要** | 工程案例教学的经典困境是"干案例"——案例被压缩为项目背景（"某市日产垃圾X吨"）+ 技术参数（"拟建焚烧厂Y吨/日"）+ 决策结论（"经论证采用Z工艺"），学生读到的是一份经过高度抽象和事后加工的"无菌报告"，而非一个有温度、有张力、有多维冲突的工程现场。本文以成都大学研究生课程《固体废弃物污染防治技术》的教学实践为案例，提出"纪实小说云驱动"的工程案例故事化重构路径。该路径以黄正文教授已公开发布的七部纪实网络小说为叙事素材云（素材母库S001-S021，凡21条教学选段），通过"五段叙事拆解法"（背景→冲突→决策→结局→启示）将真实固废管理事件转化为标准化的教学案例。文章系统论述了纪实小说作为教学素材的四个独特优势（版权清晰、叙事深度、可控取材、持续更新），并以Agent-S协同拆解流程展示了AI辅助下的案例库扩展机制。该方法为工程教育的案例教学提供了一条成本可控、版权合规、可持续迭代的新路径。 |

### 基于OBE理念的"五维错配-修复"框架教学设计——以《固体废弃物污染防治技术》为例

|---|---|

| **摘要** | 成果导向教育（OBE）理念要求课程设计从"教师教了什么"转向"学生能做什么"，但如何将抽象的课程目标转化为具体的、可操作的、逐周递进的教学活动序列，是工程教育领域长期未解的"最后一公里"问题。本文以成都大学研究生课程《固体废弃物污染防治技术》为案例，系统阐述"五维错配-修复"框架的学理建构与教学设计应用。该框架以"错配"为核心概念，构建了空间错配五维、选址正义性四维、技术错配三型、垃圾时间三维、行为错配四因、EPR错配三源、人文错配三维七个独立子框架，覆盖空间、技术、行为、人文四个认知领域，时间维度贯穿其中。子框架以递进逻辑（空间→技术→行为→人文）组织为认知加工深度逐层升高的教学序列。顶层"五维统一诊断矩阵"（空间×技术×行为×人文×时间）为学生提供了一个系统诊断固废问题的综合工具。文章构建了OBE课程目标-五维框架-八周教学内容之间的完整映射矩阵，论证了该框架作为Bloom分类学与具体工程教学内容之间的"中间层"桥梁功能。 |
