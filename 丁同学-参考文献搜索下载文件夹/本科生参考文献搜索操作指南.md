# 本科生参考文献搜索操作指南

> 适用对象：环境工程本科生，零文献检索经验  
> 执行地点：成都大学图书馆电子阅览室（或任何可访问校园网的电脑）  
> 版本：v1.0 · 2026.05.25  

---

## 第0章：行动前准备（预计5分钟）

### 0.1 你需要带什么

| 物品 | 用途 |
|------|------|
| 学生证/一卡通 | 图书馆门禁 + 登录知网/WoS |
| U盘（≥2GB空闲） | 存放下载的PDF文件 |
| 笔记本 + 笔 | 记录检索中发现的问题 |
| 学号 | 登录学校统一认证系统 |

### 0.2 你需要确认什么

在图书馆电子阅览室的电脑上，打开浏览器，依次确认以下三个网页能打开：

| 序号 | 网址 | 预期看到的画面 |
|------|------|-------------|
| 1 | `www.cnki.net` | 红色为主色调的"中国知网"首页 |
| 2 | 成都大学图书馆主页 | 能看到"外文数据库"入口 |
| 3 | `scholar.google.com` | Google学术搜索首页（备用） |

如果第2步找不到"外文数据库"，请举手找图书馆老师帮忙。就说："老师好，我想用Web of Science查英文文献，请问入口在哪？"

### 0.3 创建文件夹

在你的U盘里，新建一个文件夹，命名为你的姓名拼音首字母，例如 `ZHANGSAN`。

在这个文件夹里，新建7个子文件夹：


## 第1章：中文文献搜索（知网CNKI）

**本节目标**：学会在知网上用关键词搜到一篇论文，判断它是否相关，下载它。

### 1.1 进入知网

| 步骤 | 操作 | 你应该看到的 |
|------|------|------------|
| ① | 打开浏览器（建议Chrome或Edge）| |
| ② | 在地址栏输入 `www.cnki.net`，回车 | 红色调的"中国知网"首页 |
| ③ | 点击页面右上角"登录"→ 选"校外访问/机构登录" | 出现学校列表 |
| ④ | 搜索框输入"成都大学"→ 点击 | 跳转到成都大学统一认证页面 |
| ⑤ | 输入你的学号和密码（和选课系统一样）→ 登录 | 页面右上角显示"欢迎 成都大学" |

> **如果你卡住了**：看不到"成都大学"选项？请改用"IP登录"方式——只要你在图书馆电子阅览室的电脑上，知网会自动识别为成都大学用户。

### 1.2 第一次检索（以P1为例）

| 步骤 | 操作 | 你应该看到的 |
|------|------|------------|
| ① | 点击知网页顶部的"高级检索"（一个放大镜图标，通常在搜索框右边）| 出现多个输入框 |
| ② | 第一个输入框：粘贴"叙事教学" | |
| ③ | 第一个输入框前面的下拉菜单：保持"主题"不变 | |
| ④ | 点"+ 添加"按钮 → 新增一行 | |
| ⑤ | 第二个输入框：粘贴"工程教育" | |
| ⑥ | 第二个输入框的"与/或/非"选"AND（与）" | |
| ⑦ | 发表时间：起始"2021"，结束"2026" | |
| ⑧ | 来源类别：勾选"北大核心""CSSCI" | |
| ⑨ | 点击蓝色"检索"按钮 | 出现结果列表 |

### 1.3 判断一篇论文是否相关

浏览结果列表，对于每一条：

| 判断项 | 问自己 | 行动 |
|--------|--------|------|
| 标题 | 标题中有没有我搜索的关键词？ | 有→继续；没有→跳过 |
| 摘要 | （点击标题进入详情页）摘要是讲什么的？是关于工程教育的吗？ | 是→下载；不是→跳过 |
| 期刊 | 是不是"高等工程教育研究""大学教育"之类的教育期刊？ | 是→优质；不是→继续看 |
| 年份 | 是不是2021年之后的？ | 是→优先下载；更早的→先不下载 |

> **关键词避坑**：如果搜到的是关于"语文课上的叙事教学"——跳过！我们要的是"工程教育"中的叙事教学。

### 1.4 下载和保存

| 步骤 | 操作 |
|------|------|
| ① | 在论文详情页，找到"PDF下载"按钮 → 点击 |
| ② | 浏览器会弹出保存对话框 → 导航到你的U盘 |
| ③ | 进入 `ZHANGSAN/P1-参考文献/` 文件夹 |
| ④ | 文件名改为：`001-作者姓名-论文主标题-年份.pdf`（例如：`001-张三-叙事教学法在工程研究生课程中的应用研究-2023.pdf`）|
| ⑤ | 点击"保存" |

> **PDF打不开？** 试试"CAJ下载"——需要安装知网专用的"CAJViewer"阅读器。图书馆电脑通常已安装。

### 1.5 记录已下载文献

打开U盘里的 `文献清单.txt`，每下载一篇就添加一行：

```
P1-001 | 张三 | 叙事教学法在工程研究生课程中的应用研究 | 高等工程教育研究 | 2023 | 知网
```

## 第2章：英文文献搜索（Web of Science）

**本节目标**：学会用Web of Science搜英文论文并获取PDF。

### 2.1 进入Web of Science

| 步骤 | 操作 | 你应该看到的 |
|------|------|------------|
| ① | 打开成都大学图书馆主页 | 图书馆网站 |
| ② | 找到"外文数据库"或"电子资源"栏目 | 列表中有很多数据库 |
| ③ | 找到"Web of Science 核心合集"→ 点击 | WoS首页，白色背景+"Clarivate"标志 |
| ④ | 确认右上角有"Institutional Sign-in"或已显示"成都大学" | 如果没有，点击"Sign in"→"Institutional"→搜索"Chengdu University" |

> **如果找不到WoS入口**：求助图书馆老师。备用方案：直接用 Google Scholar（见第3章）。

### 2.2 英文关键词检索（以P1为例）

| 步骤 | 操作 |
|------|------|
| ① | 在WoS首页搜索框右侧，选择"All Databases"或"Web of Science Core Collection" |
| ② | 搜索框粘贴：`"narrative pedagogy" AND "engineering education" AND "graduate"` |
| ③ | 点击"Search" |
| ④ | 结果页左侧"Publication Years"→ 勾选 2021-2026 → 点击"Refine" |
| ⑤ | 左侧"Document Types"→ 勾选"Article"→ 点击"Refine" |
| ⑥ | 右上角"Sort by"选"Citations: highest first" |

### 2.3 判断英文论文是否相关（快速阅读法）

不用读懂整篇！只读以下部分：

| 读哪里 | 找什么 | 用时 |
|--------|--------|------|
| Title | 有没有我搜索的关键词？ | 5秒 |
| Abstract（摘要） | 这篇论文是讲什么的？跟我的主题有关吗？ | 30秒 |
| Keywords | 列出了哪些关键词？有没有我认识的？ | 10秒 |
| Journal | 是不是教育类期刊？（如Journal of Engineering Education） | 5秒 |

> **如果摘要完全看不懂**：不用勉强。换下一篇。你的任务是找"相关"的文献，不是翻译文献。

### 2.4 获取全文

WoS本身不提供PDF下载，需要转到出版社网站：

| 步骤 | 操作 |
|------|------|
| ① | 点击论文标题进入详情页 |
| ② | 找到"Full Text"或"Publisher's Website"链接 → 点击 |
| ③ | 跳转到出版社网站（如Taylor & Francis, Elsevier, Springer）|
| ④ | 如果页面显示"Download PDF"→ 点击下载 → 保存到 `P1-参考文献/` |
| ⑤ | 如果页面要求"Pay"或"Login"→ 记录DOI（如`10.1002/j.2168-9830.2022.tb01001.x`）→ 放到"待获取"清单 |

> **DOI是什么？** 一篇论文的唯一身份证号。格式如`10.xxxx/xxxxx`。记下DOI，后面可以用其他方式获取全文。

### 2.5 命名英文文献文件

```
P1-E001-Smith-Narrative pedagogy in graduate engineering education-2022.pdf
```

## 第3章：备用方案（Google Scholar + Sci-Hub）

**本节目标**：当WoS无权限或找不到全文时，用Google Scholar和Sci-Hub获取。

### 3.1 Google Scholar搜索

| 步骤 | 操作 |
|------|------|
| ① | 打开 `scholar.google.com` |
| ② | 搜索框粘贴英文关键词，例如：`"narrative pedagogy" "engineering education"` |
| ③ | 左侧"Custom range"→ 输入 2021-2026 |
| ④ | 浏览结果，点击标题阅读摘要 |

### 3.2 用DOI获取全文（最可靠的方法）

如果你只拿到一个DOI，按以下步骤获取全文：

| 步骤 | 操作 |
|------|------|
| ① | 打开浏览器 → 地址栏输入 `sci-hub.se` → 回车 |
| ② | 在页面中间的输入框粘贴DOI → 点击"Open" |
| ③ | 如果成功，会直接显示PDF → 点击右上角下载图标保存 |

> **如果Sci-Hub打不开**：试试备用域名 `sci-hub.st` 或 `sci-hub.ru`。学校图书馆的"文献传递"服务是正规替代方案（问图书馆老师"怎么申请文献传递"）。

## 第5章：下载后的整理工作

### 5.1 文件命名规范

每篇下载的论文必须立即重命名，格式为：

```
{论文编号}-{序号}-{第一作者姓氏}-{论文主标题前10个词}-{年份}.pdf
```

示例：

| 编号格式 | 说明 |
|----------|------|
| `P1-001-张三-叙事教学法在工程研究生课程中-2023.pdf` | 中文论文 |
| `P1-E001-Smith-Narrative pedagogy in graduate-2022.pdf` | 英文论文（E=English） |

> **为什么必须立即重命名？** 知网下载的文件名通常是乱码或数字，不重命名就无法分辨。

### 5.2 填写文献清单

打开 `文献清单.txt`，按以下模板每下载一篇添加一行：

```
P1-001 | 张三 | 叙事教学法在工程研究生课程中的应用研究 | 高等工程教育研究 | 2023 | 知网
P1-E001 | Smith J | Narrative pedagogy in graduate engineering education | J Eng Educ | 2022 | WoS
```

### 5.3 最终文件夹结构

完成后，你的U盘应该是这样的：

```
ZHANGSAN/
├── P1-参考文献/
│   ├── P1-001-张三-叙事教学法-2023.pdf
│   ├── P1-002-李四-情境学习-2022.pdf
│   ├── P1-E001-Smith-Narrative pedagogy-2022.pdf
│   ├── P1-E002-Jones-Situated learning-2021.pdf

│   └── ...（同理）
├── ...

│   └── ...

```

## 第6章：质量自检清单

### 每篇论文完成搜索后，对照以下清单逐项打勾：

| # | 检查项 | 通过标准 | ☐ |
|---|--------|---------|---|
| 1 | 来源期刊 | 是否为SSCI/SCI/CSSCI/北大核心期刊？如果不是，这篇不能用 | ☐ |
| 2 | 发表年份 | 是否为2021-2026年发表的？（经典理论著作除外） | ☐ |
| 3 | 标题相关 | 论文标题是否包含搜索关键词？ | ☐ |
| 4 | 摘要相关 | 摘要内容是否与论文主题直接相关？ | ☐ |
| 5 | 全文完整 | PDF是否能正常打开？页码是否完整？ | ☐ |
| 6 | 无重复 | 是否与已下载的其他论文不重复（查标题和作者）？ | ☐ |
| 7 | 数量达标 | 中文≥2篇 + 英文≥3篇？ | ☐ |
| 8 | 文件已命名 | 文件是否已按规范重命名？ | ☐ |
| 9 | 已录入清单 | 是否已添加到`文献清单.txt`？ | ☐ |

## 第7章：常见问题与解决方法

| 问题 | 可能的原因 | 解决方法 |
|------|----------|----------|
| 知网提示"未登录" | 没有通过学校入口登录 | 关掉页面，重新从图书馆主页→知网入口进入 |
| 知网搜不出来 | 关键词太窄 | 减少一个关键词，或者尝试"同义词"（如"案例教学"换成"案例法"） |
| 知网搜到太多 | 关键词太宽 | 加一个AND条件，或者限制"来源类型"为"核心期刊" |
| WoS搜索结果为0 | 英文关键词拼写错误 | 逐词检查，特别是引号是否成对 |
| WoS有结果但无全文 | 成都大学未订购该期刊 | 记录DOI → 用Sci-Hub试 → 不行则用"文献传递" |
| Sci-Hub打不开 | 域名被屏蔽 | 试备用域名：`sci-hub.st` / `sci-hub.ru` / `sci-hub.se` |
| PDF下载后打不开 | 文件损坏 | 删除重新下载 |
| CAJ格式打不开 | 未安装阅读器 | 在知网首页下载"CAJViewer"→安装→再打开 |
| 不知道该下载哪篇 | 难以判断相关性 | 优先下载"摘要中出现你的全部关键词"的论文 |
| U盘空间不足 | 通常每篇PDF约0.5-2MB，35篇约20-70MB | 任何≥1GB的U盘都够用 |
