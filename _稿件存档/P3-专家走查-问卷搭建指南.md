# P3专家走查问卷 · 一键搭建指南

## 方式一：腾讯问卷（推荐，免费，微信可直接打开）

### 10分钟搭建步骤：

```
1. 打开 https://wj.qq.com/ → 微信扫码登录
2. 点击「创建问卷」→「空白创建」
3. 标题填：P3论文专家走查 · 设计原则评估
4. 按下方内容逐题添加
```

---

### 题目清单（逐题复制粘贴）

**题目1** — 单选
```
题目标题：您的角色类型（可多选）
题型：多选题
选项：
  工科研究生课程教师
  教育技术/信息化教学方向研究人员
  课程设计/教学改革研究者
  在读研究生（工科）
  其他
```

**题目2** — 单选
```
题目标题：您对AI教育工具的熟悉程度
题型：单选题
选项：
  非常熟悉（日常使用多种AI工具辅助教学/学习）
  比较熟悉（偶尔使用ChatGPT/文心一言等）
  一般了解（知道但不太使用）
  不太熟悉
```

**题目3-9** — 矩阵量表（7题一组）
```
题目标题：请根据阅读评估材料后的第一直觉打分（1=完全不同意，7=完全同意）
题型：矩阵量表（Likert 7点）

量表行（7行）：
  Q1：三条原则的表述清晰、易于理解
  Q2：三条原则完备——覆盖了Agent角色设计的主要考量
  Q3：三条原则具有可操作性——能指导实际的Agent角色设计
  Q4："镜像而非简化"原则相对于"简化版学生工具"思路，具有差异化价值
  Q5：三条原则与引言中三个结构缺陷对应准确
  Q6：该设计方法论具有向其他工程课程迁移的潜力
  Q7：整体而言，该设计方案的学术发表价值达到CSSCI期刊水平

量表列（7列）：
  1  2  3  4  5  6  7
```

**题目10** — 开放题
```
题目标题：任一条原则中，是否存在理解困难或逻辑跳跃之处？（可选）
题型：文本题（多行）
```

**题目11** — 开放题
```
题目标题：您是否有任何补充建议？（可选）
题型：文本题（多行）
```

### 发布

```
点击「发布」→ 获取链接/二维码 → 分享给专家
```

## 方式二：Google Form（英文界面，适合有 Google 账号的）

1. 打开 https://forms.google.com
2. 新建表单，标题同上
3. 按上方内容逐题添加（操作逻辑类似）

## 方式三：直接发评估材料 + 让专家微信回复分数

如果实在不想搭问卷，最简方案：

把 `P3-专家走查-邀请函与评估材料.md` 发过去，末尾加一句：

```
请直接回复以下7项的分数（1-7分）：
Q1=__  Q2=__  Q3=__  Q4=__  Q5=__  Q6=__  Q7=__
如有建议请注明：______
```

然后手工汇总。
