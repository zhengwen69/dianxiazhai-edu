## 第一章：格式铁律

### 第1条（字数）
全文 **8000-12000 字**。违宪等级：**一级**。

### 第2条（摘要）
中文摘要 **200-300 字**。英文摘要对应。违宪等级：**一级**。

### 第3条（关键词）
**3-8 个**，中英对照。违宪等级：**二级**。

### 第4条（图表标题）
图表须中英双语标题。格式："图N 中文 / Fig.N English"、"表N 中文 / Table N English"。违宪等级：**一级**。

### 第5条（参考文献格式）
GB/T 7714-2015 顺序编码制。**引文须按首次出现顺序编号**，每条须有正文引用。**必须含 [J][M][Z][EB/OL] 类型标识**及卷期页码。违宪等级：**一级**。

### 第6条（引文数量）
**≥15 条**，英文比例≥30%。违宪等级：**二级**。

### 第7条（匿名化）
正文匿名，作者信息另页存放。违宪等级：**一级**。

### 第8条（页面设置·排版细则—v1.1强化）

| 要素 | 规范 | 违宪等级 |
|------|------|:--:|
| 纸张 | A4 | — |
| 栏数 | **双栏**，栏距 20pt | 一级 |
| 页边距 | 上下 2.5cm，左右 2.0cm | 二级 |
| 正文 | 宋体 10.5pt（五号），1.5 倍行距，首行缩进 0.74cm | 一级 |
| 引文上标 | 正文 [N] 格式须为 **8pt 上标** | 一级 |
| 引号 | 正文中文区**全部全角**（\u201c\u201d），禁用半角 `"` | 一级 |
| 参考文献字体 | **六号 7.5pt**，行距 1.2 倍，无缩进 | 一级 |
| 参考文献中英文字体 | 中文宋体，英文 Times New Roman | 二级 |
| 图宽度 | **3.0 英寸**（适配双栏） | 二级 |
| 图标题 | 宋体 9pt 灰色居中 | 二级 |
| 表格式 | **三线表**（顶线 1.5pt / 栏目线 0.75pt / 底线 1.5pt），无竖线 | 一级 |
| 表字体 | 宋体六号 7.5pt，表头加粗居中，单倍行距 | 一级 |
| 作者信息 | 另页提交，含出生年/性别/职称/研究方向/Email/ORCID | 一级 |

## 第二章：自动化质控机制

### 第9条（版本安全）
任何修改前版本自增：`python version_bump.py P1-A轨`。违宪等级：**一级**。

### 第10条（去AI味）
投稿前运行深度 AI 扫描，清零"为...提供/通过...实现/不仅...而且/该框架/本文旨在/扮演...角色"等 20+ 模式。违宪等级：**一级**。

### 第11条（引文核查）
逐条验证参考文献真实性。DOI 校验英文文献，知网匹配中文文献。高风险条目自动删除。违宪等级：**一级**。

### 第12条（自检）
投稿前运行 `check_all.py`，A01-A07 全部 PASS 方可投稿。违宪等级：**一级**。
