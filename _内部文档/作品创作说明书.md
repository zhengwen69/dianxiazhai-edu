# 作品创作说明书

## 作品名称
《叙事·门户·智能体——"叙·框·境·创"工程专业硕士研究生教学法》

## 作品类型
文字作品（学术专著）

## 作者
黄正文（笔名：点暇斋、点暇斋主），成都大学环境工程系教授

## 创作完成日期
2026 年 5 月

## 作品规模
正文约 20 万字，10 章 + 4 附录，约 450 页

## 创作意图
本书旨在系统阐述作者原创的"叙·框·境·创"（Narrative-Framework-Scene-Creation, NFSC）四维研究生教学法。该教学法系作者在成都大学环境工程专业研究生核心课程《固体废弃物污染防治技术》的教学实践中，经过三轮系统性改革（2023-2026）自主研发而成。

## 创作过程
1. **实践积累（2003-2023）**：作者深耕工科研究生教学凡二十年，积累大量课堂教学经验与工程咨询案例。
2. **叙事素材创作（2019-2026）**：作者以"点暇斋"为笔名，在 QQ 阅读、17K 文学平台连载纪实小说七部（《龙栖湾》《青龙湖》《杨柳坝与刘家湾》《三多里巷》等），累计约 200 万字，为教学法提供叙事素材云。
3. **教学法形成（2023-2026）**：课程经 v1.0（传统讲授）、v2.0（错配-修复框架）、v3.0（四维教学法 + 八周思维重塑）三轮迭代。
4. **专著撰写（2025-2026）**：系统总结四维教学法的理论建构、分析框架、数字基座、AI Agent 协同、课程思政融入及八周实践全过程。

## 独创性
1. 原创"叙·框·境·创"四维递进教学模型（NFSC）
2. 原创"五维错配-修复"分析体系（七大子框架 + 五维统一诊断矩阵）
3. 原创纪实小说云驱动案例方法（五段叙事拆解法 + S001-S021 素材母库）
4. 原创"教学-学习"双重数字门户架构（纯静态 HTML/CSS/JS + GitHub Pages 零费部署）
5. 原创九-Agent 协同教学体系与教师→学生双向角色映射法
6. 原创"双线叙事思政"育人模式与"看不见的思政"原则
