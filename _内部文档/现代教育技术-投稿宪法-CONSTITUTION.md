## 第一章：格式铁律

### 第1条（字数）

全文（含图表、参考文献、英文摘要）**8000-10000 字**，不低于 6000 字。违宪等级：**一级**。

### 第2条（摘要）

中文摘要 **200-300 字**，第三人称，含研究目的、方法、成果、结论。英文摘要对应。违宪等级：**一级**。

### 第3条（关键词）

**3-8 个**，中英对照。违宪等级：**二级**。

### 第4条（图表标题）

所有图表须有**中英双语标题**。格式："图N 中文 / Fig.N English"、"表N 中文 / Table N English"。违宪等级：**一级**。

### 第5条（页面设置）

A4 纸，上2.3cm/下2.3cm/左1.8cm/右1.7cm，双栏排版。正文小五号（9pt）宋体。违宪等级：**二级**。

### 第6条（匿名化）

正文及文件名不得出现作者姓名、单位、基金项目、作者简介，另页存放。违宪等级：**一级**。

### 第7条（参考文献格式）

GB/T 7714-2015 顺序编码制。每条文献必须在正文有对应引用，不得有"有文无引"。**引文须按首次出现顺序编号为 [1][2]...[N]**——非顺次编号即为违宪。违宪等级：**一级**。

### 第8条（标题层级）

正文标题：一 →（一）→ 1 →（1）→ ①。违宪等级：**二级**。

### 第9条（中图分类号）

须标注中图分类号与文献标志码。违宪等级：**三级**。

## 第二章："六有"审查自检清单

编辑部明文以"六有"为审稿标准。投稿前须逐项自检：

| # | 标准 | P3 验证项 | 通过条件 |
|:--:|------|---------|------|
| 1 | **有价值** | 多智能体协同体系提供可复用方法论 | 结论段明确陈述实践价值 |
| 2 | **有创新** | "教→伴学"双向映射为原创贡献 | 摘要/结论明确差异化创新点 |
| 3 | **有观点** | 三原则+双向映射表为核心观点 | 每观点有理论依据+案例支撑 |
| 4 | **有综述** | 18条引文覆盖 AI/Agent/LLM | ≥15条、无有文无引 |
| 5 | **有研究** | 试用反馈+系统对比表 | 有数据或对比证据 |
| 6 | **有规范** | GB/T 7714+双语图表+匿名化 | 格式逐项通过 |

违宪等级：任一项不达标为**一级**——不得投稿。

## 第三章：自动化质控机制（v1.1 新增）

### 第10条（版本安全铁律）

任何修改前必须执行版本自增：
```
python version_bump.py P3-AI-Agent-排版稿
```
产出新版本 `.md` 文件，后续 `.docx`/`.pdf` 对应新版本生成。**禁止覆盖旧版本三件套**。版本命名：`_v{M}.{N}`，M 大版本，N 小版本。违宪等级：**一级**。

### 第11条（自检脚本）

投稿前必须运行：
```
python check_all.py P3-AI-Agent-排版稿_vX.Y.md
```

| # | 检查项 | 通过标准 |
|:--:|------|------|
| A01 | 引文顺次编号 | [1][2]...[N] 按首次出现排列 |
| A02 | 引文上标格式 | docx 导出后上标 |
| A03 | 摘要字数 | 200-300 字 |
| A04 | 图表双语标题 | 全图/表含英文 |
| A05 | 正文匿名化 | 作者信息在另页 |
| A06 | 参考文献格式 | GB/T 7714 |
| A07 | 总字数 | 8000-10000 字 |

任一项 FAIL 不得投稿。违宪等级：**一级**。

### 第12条（引文重编号）

若引文未按首次出现顺序排列，运行：
```
python reorder_refs.py
```
脚本自动扫描顺序→建立旧→新映射→替换引用→重排参考文献表。违宪等级：**一级**。

## 第四章：投稿流程 SOP

```
1.注册作者 → 2.下载模板 → 3.排版适配 → 4.提交匿名稿件 → 5.外审(3月) → 6.退修(2月) → 7.录用缴费 → 8.出版(4-6月)
```

**第1步**：在 https://xjjs.cbpt.cnki.net/ 注册作者账号。

**第2步**：从"下载中心"获取论文模板（.doc 格式）。

**第3步**：将 P3 填入模板——中文题名≤20字、作者信息另页、中英文摘要+关键词、正文双栏、图表中英双语、参考文献 GB/T 7714。

**第4步**：提交匿名稿件。系统分配稿件编号。

**第5-8步**：外审→退修→录用→出版。
