🧪 论文系列 (Thesis Track) — 4 题
论-1 环境工程专业固废课程中纪实小说辅助学习的认知体验初探
🧪 毕业论文 质性研究 自我民族志 + 半结构化访谈 5-8人

学生人设：固废课堂的学习者反思者。以自身8周学习日志和5-8名同学访谈数据，探索纪实小说作为辅助学习材料在认知过程中扮演的角色。已有素材：7部长篇小说全文、课程大纲、NFSC教学法文档。

论-2 环境工程专业固废课程学习门户可用性评价研究
🧪 毕业论文 量化研究 SUS/TAM 问卷 15-20份 SPSS

学生人设：学习系统的用户测试者。定制SUS可用性量表+TAM技术接受模型问卷，发放15-20份，完成统计分析。已有素材：固废课程学习子门户、问卷模板、Agent辅助系统。

论-3 环境工程专业固废课程 NFSC 教学法要素在迁移场景中的呈现比较分析
🧪 毕业论文 定性比较分析 多案例 6卡素材

学生人设：跨场景的观察比较者。从6个迁移场景中系统提取NFSC四要素，构建比较矩阵，归纳要素保留与变异的规律。已有素材：迁移探索6卡详情页、教学日志、课程设计文档。

论-4 环境工程专业固废课程多语种学习门户无障碍性研究
🧪 毕业论文 启发式评估 Nielsen 10原则 8语种

学生人设：国际化适配的评估者。对照Nielsen 10项启发式原则逐页评估8语种门户，核对翻译完整性与准确性。已有素材：8语种HTML门户、i18n JSON语言包、Nielsen评估量表模板。

🔧 设计系列 (Design Track) — 4 题
设-1 环境工程专业固废浏览页——迁移探索 Pending 详情页的设计
🔧 毕业设计 HTML + CSS + JS i18n 国际化 2个页面

学生人设：前端设计者。补全迁移探索6卡体系中2个Pending卡片的详情页，遵循现有深色主题模板风格，接入8语种i18n框架。已有素材：迁移探索枢纽页模板、4个已完成的详情页源码、i18n.js框架。

设-2 环境工程专业固废浏览页——NFSC 方法论时间线的设计
🔧 毕业设计 HTML + CSS + JS 时间线交互 i18n

学生人设：信息架构设计者。梳理NFSC方法论从2023年诞生到2026年6卡迁移的完整时间节点，设计纵轴交互时间线页。已有素材：6卡素材、教学日志、课程大纲、P1论文。

设-3 环境工程专业固废浏览页——小说-教学法映射可视化的设计
🔧 毕业设计 HTML + CSS Grid + JS 交互矩阵 i18n

学生人设：内容可视化设计者。构建"7部小说×8周课程×4种错配"三维映射矩阵，实现支持按类型筛选的交互式可视化页。已有素材：7部长篇目录、课程大纲、错配-修复框架文档。

设-4 环境工程专业固废课题组育人成果展示浏览页的设计
🔧 毕业设计 HTML + CSS Grid + JS 作品集 分类筛选 i18n

学生人设：作品集设计者。汇聚课题组毕设/大创/专利/竞赛等育人成果，设计支持标签分类筛选与详情展开的交互式作品集展示页。已有素材：毕设归档材料、大创证书、专利清单、竞赛成果。
