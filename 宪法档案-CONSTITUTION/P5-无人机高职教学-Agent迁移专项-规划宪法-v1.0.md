# P5 规划宪法：无人驾驶航空器（UAV）高职教学 · Agent 迁移专项

> 适用范围：NFSC+Agent 方法论从固废研究生课程向无人驾驶航空器（UAV）高职高专课程的迁移应用
> 版本 v1.0 · 2026-06-01
> 最高准则：凡规划、实施、修改之行为，均须符合本宪法之规定。

---

## 序言：总纲与宗旨

本宪法为谢泽宇（第一作者，西那瓦国际大学博士生）将其在黄正文教授课题组所参与的 NFSC（叙·框·境·创）教学法、Agent 智能体辅助教学系统、双门户架构从固体废弃物研究生课程向无人驾驶航空器（Unmanned Aerial Vehicle, UAV）高职高专课程进行系统性迁移的规划总纲。

**核心命题**：同一套方法论，能否从研究型研究生课程迁移至技能型高职课程？迁移过程中需要保留什么、舍弃什么、新增什么？

**学科全称对照**：

| 简称 | 全称（中文） | Full Name (English) |
|------|------|------|
| 无人机 | 无人驾驶航空器 | Unmanned Aerial Vehicle (UAV) |
| 高职 | 高等职业教育 | Higher Vocational Education |
| 固废 | 固体废弃物 | Solid Waste |
| Agent | 智能体辅助教学系统 | Agent-Assisted Teaching System |

## 第一章：背景与定位

### 1.1 迁移背景

黄正文教授课题组已在固体废弃物污染防治技术研究生课程中验证了 NFSC 教学法 + Agent 智能体系统 + 双门户架构的有效性。谢泽宇作为课题组成员，目前在高职院校担任 UAV 应用技术专业专职教师，已授课 10 学期（2021–至今），涵盖 10 门 UAV 课程。

谢泽宇本硕均非 UAV 专业背景，其自身从跨专业学习者到 UAV 教师的成长历程，本身就是 NFSC+Agent 方法论可迁移性的活案例。在此基础上，将整套方法论系统性地迁移至其所授课的高职学生群体，构成第二次迁移验证。

### 1.2 与现有论文的关系矩阵

| 论文 | 核心论点 | 与 P5 关系 |
|------|------|------|
| P1 (EJEE) | NFSC 教学法在固废研究生课程的理论建构 | P5 的理论源头 |
| P2 (KJSS) | 双门户架构设计与多语部署 | P5 的技术基建 |
| P3 (现代教育技术) | Agent 教师↔学生双向角色映射 | P5 的 Agent 模板 |
| P4 (SAJE) | 谢泽宇 UAV 自学跨学科教师成长 | P5 的个人叙事基础 |
| **P5 (拟新建)** | **NFSC+Agent 从固废研究生→UAV 高职的系统性迁移实证** | **本宪法** |

P5 与 P4 的差异化：P4 聚焦谢泽宇**个人自学**历程（autoethnography），P5 聚焦谢泽宇作为教师的**教学实践**成效（学生数据 + 课程设计 + 同行复制）。

## 第二章：五层迁移模型

黄之固废（研究生层次）向谢之 UAV（高职高专层次）的五层系统性迁移：

### 2.1 叙（Narrative）：从纪实小说到碎片化教案

| 维度 | 固废课程（黄） | UAV 课程（谢） | 迁移策略 |
|------|------|------|------|
| 核心素材 | 7 部已出版纪实网络小说 | 10 学期讲义 + 故障案例库 + 行业事故报告 | 碎片化教案替代长篇小说选段 |
| 素材特征 | 真实工程场景、人物命运交织 | UAV 飞行事故、执照考试真题、企业案例 | 行业故事替代文学叙事 |
| 素材形式 | 长篇文学文本 | 短篇案例分析（300–500 字/案） | 降低认知负荷 |
| 版权状态 | © 版权登记 | © 待整理登记 | 建立素材版权意识 |

### 2.2 框（Framework）：从四重错配到 UAV 知识域拆解

| 固废框架 | → | UAV 框架 |
|------|:--:|------|
| 空间错配 | → | 空域规划与飞行安全管理 |
| 技术错配 | → | 机型选型与作业场景匹配 |
| 行为错配 | → | 飞手操作规范与应急决策 |
| 人文错配 | → | 无人机隐私伦理与社区接受度 |
| 时间错配 | → | 设备维护周期与电池寿命管理 |

### 2.3 境（Scene）：从学术场景到执照考核场景

| 固废场景 | → | UAV 场景 |
|------|:--:|------|
| 填埋场选址公众听证会 | → | 飞行前安全检查沙盘推演 |
| 脱硫石膏技术路线论证会 | → | 多机型作业方案比选 |
| 渗滤液运营危机应对 | → | 飞行失联应急处理模拟 |
| 模拟法庭 | → | 无人机事故责任认定 |
| 碳交易谈判 | → | 无人机服务报价谈判 |

### 2.4 创（Create）：从学术产出到技能考核产出

| 固废产出 | → | UAV 产出 |
|------|:--:|------|
| 课程论文 / 课程设计 / 教改论文 | → | 技能考核合格证 / UAV 执照 / 竞赛获奖 |
| 五维统一诊断矩阵 | → | 飞行作业综合评估表 |
| 思维重塑报告 | → | 飞行实习总结报告 |

### 2.5 Agent 智能体角色重映射

| Agent | 固废课程角色 | → | UAV 高职课程角色 |

| Agent 4 作业课设 | 固废工程课设 | → | 航线规划/维修课设/执照模拟题 |

## 第三章：素材库建设方案

### 3.1 素材来源

谢泽宇自 2021 年至今在高职院校讲授 UAV 课程的原始教学档案，涵盖 10 学期、10 门课程。

### 3.2 组织架构

```
谢泽宇UAV素材库/
├── 01-航空数据分析与处理/
├── 02-无人机操控与航迹规划/
├── 03-传感器与检测技术/
├── 04-机械设计基础/
├── 05-无人机模拟飞行/
├── 06-无人机地面站/
├── 07-无人机飞行操控原理与应用/
├── 08-农用植保无人机技术/
├── 09-无人机电力巡检技术/
├── 10-UAV-Primary-Mission-Flight/
├── 共通/
│   ├── 教学大纲/
│   ├── 学期总结/
│   └── 学生成绩/
└── 课题/
    ├── 省级教改项目/
    └── 校级教改项目×2/
```

### 3.3 归档要求

- 每门课程至少包含：教学大纲、代表性讲义（3–5 份）、考核标准、学生代表性作品（匿名）
- 共通文件含：10 学期教学日志、学生成绩分布、课程满意度数据（如有）
- 课题文件含：立项书、中期报告、结题证明
- 所有学生信息需匿名化处理
